# 🌐 AWS Static Website Hosting (S3 + CloudFront)

## 🚀 Project Overview
This project demonstrates how to deploy a static website using AWS cloud services with high performance and security.

The website is hosted on Amazon S3 and delivered globally using Amazon CloudFront CDN with HTTPS enabled.

## 🌍 Live Demo

👉https://d3d6j3ncte1dxx.cloudfront.net/

---

## 🎯 Objectives
- Deploy a static website on cloud
- Improve performance using CDN
- Secure content delivery using HTTPS
- Implement access control using AWS IAM

---

## 🛠️ Technologies Used
- AWS S3
- AWS CloudFront
- AWS IAM
- HTML, CSS, JavaScript

---

## ⚙️ Implementation Summary
- Created and configured an S3 bucket for static hosting
- Uploaded website files (HTML, CSS, JS)
- Enabled static website hosting
- Configured access permissions
- Integrated CloudFront CDN for fast delivery
- Enabled HTTPS for secure communication

## 🔐 Security Features
- HTTPS enabled via CloudFront
- Secure access configuration using AWS policies
- CDN-based protection against common threats
